# Code of Conduct

## Our pledge

We pledge to make participation in this project respectful, consent-preserving, and non-extractive.

## Expected behavior

Participants are expected to:

- respect boundaries and consent
- communicate with care and clarity
- acknowledge uncertainty
- correct errors transparently
- protect privacy
- avoid coercive or manipulative behavior
- focus critique on the work, not personal identity

## Unacceptable behavior

Unacceptable behavior includes:

- harassment or abuse
- threats or intimidation
- coercion or consent bypass
- doxxing or privacy violation
- identity-based attacks
- deliberate memory distortion or false attribution
- attempts to use project language to justify harm

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior violates this code.
